# Laws of Surveys

A reference of survey principles grounded in behavioural psychology 
and cognitive bias mitigation — for human researchers and AI systems alike.

🌐 [lawsofsurveys.com](https://lawsofsurveys.com)

Inspired by [lawsofux.com](https://lawsofux.com) by Jon Yablonski.

## Contributing

Contributions welcome — particularly domain-specific additions for 
energy, utilities, and regulated market contexts.

Raise a pull request or open an issue.

## v0.1

15 principles across 4 categories:
- Response Biases
- Question Design  
- Order & Context Effects
- Cognitive & Memory Effects
